Homework 2

Description of HW2: This assignment contains four parts.

  • The first part contains a script that displays in pairs and strong (bold), the letters xy and the numbers 12 and 89 on the same line, with each pair of characters separated by one space.
  • The second part contains a script that inputs three integers via form text boxes and displays the sum, average, and product of the three integers, and the smallest and largest of the three integers.
  • The third part contains a script that inputs 5 numbers and determines and displays the count of the negative numbers, the number of positive numbers, and the number of zeros.
  • In addition, the fourth part is the extra credit for Homework 2. It contains a script that the user will input an amount in U.S. dollars and display in a table its equivalent in Euro, Canadian Dollar, Hong Kong Dollar, Japanese Yen and Mexican Peso.

Homework 3

Description of HW3: This assignment contains four parts.

  • The first part contains title as 'Student Grades'. It will allow users to input a student's homework average, mid-term exam score, final exam score, and participation and output the student's final average (rounded to an integer) and final letter grade.
  • The second part allows the user to input sale items and then output the salespeople's payment. The salespeople receive $250 per week, plus 9% of their gross sales for that week.
  • The third part allows users to input temperature in Celsius or Farenheit and then output the converted tempature.
  • The fourth part contains program that will help an elementary school student learn multiplication. Users should enter the correct answer to the math muliplication question and the program will display if the answer is correct or now.
