SEO Suggestions

  1. We need to remove anything that slows down the website loads. According to the Amazon research, every second delay of their website loads, they would lose 1% of their revenus.

    Ways to Increase Loading Speeds:

    • Using Google PageSpeed to test out loading speeds and found out what slow down website loads.
    • Using Google AMP framework to load mobile website super fast.
    • Using CDN(content delivery network) which puts our websites all over the world.
    • Using WordPress or a plugin W3 Total Cache to cache your website.
    • For more information, please check "How to Speed Up Your Website".
  2. We should link out to relevant and authoritative content pages, which are some high-quality sites and sources, because these links build the fundamental part of smart search engine optimization strategy. Moreover, we should also encourage other trustworthy sites to link to our sites.
  3. We should first define our search engine optimization goals clearly.
    • Acoording to these goals, we need to use tools like Google Analytics, Google Search Console to measure, collect, analyze and report of web data and to understand and optimize web usage.
    • Check out following Analytics Website:
      • Google Analytics
      • Crazy Egg
      • Google Website Optimizer
  4. To increase traffic, we should focus more on how people engage with a particular brand, product and service.
  5. We should write unique and relevant meta descriptions for every page.

    • Characteristics of A good meta description:

      • Keep it up to 155 characters
      • Use Active voice and make it actionable
      • Include a call-to-action
      • Use focus keyword
      • Show specification
      • Make sure it mataches the content of the page
      • Make it unique
    • For example: Amazon Website: Free delivery on millions of items with Prime. Low prices across earth's biggest selection of books, music, DVDs, electronics, computers, software, apparel ...
    • For more detail: please check 'How to create the right meta description'.
  6. Use a Simple, Readable URL Structure
    • Include one primary keyword in a page URL
    • Using hyphens indtsead of underscores in URLs
    • Choose a static URL address instead of a dynamic one
    • Keep a URL address brief, descriptive and relevant.
    • For more information, please check "35 Tips That Make SEO Work For Your Website".
