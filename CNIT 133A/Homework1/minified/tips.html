Optimizing Images for the Web

To optimize images for the main page, I changed all png and jpg images to webp image format. According to the Performance & Organization, the webp image format is generally 26% smaller in size when compared to png format and around 30% smaller in size when compared to jpeg images.

Before optimizing the images file, its size is 532.31 kb. After converting all images to webp image format, the size of images file is 397.79 kb. Therefore, the overall size of images file are decreased by 25%.
